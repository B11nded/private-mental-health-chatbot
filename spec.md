## Project Specification

### Goal

"The goal of my chatbot is to create a private conversational mental health resource that can be used to talk about sensitive topics and serious issues."

### MVP Features

1. Conversational back-and-forth both through text and voice
2. Easy to use UI that is simple and easily understandable 
3. AI makes sure to alert the user they are not a health professional and explains the proper routes to take if asked
4. Careful consideration to prevent the AI from storing any information long-term data is only stored temporarily to get a result from openAI

### Success Criteria

- [  ] Feature one works and communicates effectively through both text and voice
- [ ] Projects UI is effective, while being simple to understand 
- [ ] AI makes it clear they are not a health care professional and gives out resources you can contact.
- [ ] Code is clean and well organized into modules
- [ ] README is complete and professional
